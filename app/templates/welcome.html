About This Analyzer

This sentiment analyzer was built using the IMDB Movie Reviews Dataset with over 50,000 labeled reviews. The application uses Natural Language Processing (NLP) techniques including:

  • Text Preprocessing: Tokenization, lemmatization, and stop word removal
  • Feature Extraction: TF-IDF vectorization for numerical representation
  • Multiple Models: Ensemble methods combining Logistic Regression, SVM, and Naive Bayes
  • Rule-Based Overrides: Enhanced detection of strong sentiment patterns
